Goal Setting

Command Line

02/21/2018
  • Learn what Terminal is(where to find it, how to launch, and what to use for)
  • Learn the basic and most common commands
  • Practise the commands you learned, create some fake actions to exercise

My goals were met today. That dark scary terminal screen is actually not that difficult to learn, at least the basics. Learned how to launch it only using the keyboard, and installed some cool packages which makes it a little easier to read. The break-out session with Casey was really helpful. We learned the most common commands at the break-out. After feeling a little comfortable, I started to play with it a little. I created a lot of folders and files, moved them around, etc. Practising like this was super helpful. But one thing I am going to do tomorrow is to read the the tutorials first before I jump directly in it. So, maybe I can write a more specific goal, saying; '- Watch some videos about the basic Command Line commands'.
